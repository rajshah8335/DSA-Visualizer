Queue Theory

A Queue is a linear data structure that follows the FIFO (First In First Out) principle. Elements are added at the rear and removed from the front.

Types:

  • Simple Queue: Linear insertion and deletion.
  • Circular Queue: The last position stays connected to the first.
  • Priority Queue: Elements are served based on priority.

C++ Implementation

#include <queue>
#include <iostream>
using namespace std;

int main() {
    queue<int> q;
    q.push(10);
    q.push(20);
    cout << q.front(); // 10
    q.pop();
    return 0;
}
