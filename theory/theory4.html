Queue Data Structure

Detailed explanation, analysis, and implementation.

What is a Queue?

A Queue is a linear data structure that follows the FIFO (First In, First Out) principle. This means that the first element added to the queue is the first one to be removed.

Think of a line of people waiting for a bus: the first person to get in line is the first person to get on the bus.

Visual Representation

OUT

20

Types of Queues

1. Simple Queue

Standard FIFO structure. Insertion at rear, deletion at front.

2. Circular Queue

The last position is connected back to the first position to make a circle, overcoming the limitation of wasted space in standard array implementation.

3. Priority Queue

Elements are associated with priority. Element with highest priority is served first.

4. Deque (Double Ended Queue)

Insertion and deletion can be performed from both the front and rear ends.

Advantages & Disadvantages

Advantages

  • Efficient Data Management: Useful for managing data in arrival order.
  • Fast Operations: Enqueue and Dequeue are O(1).

Disadvantages

  • Random Access: Typically not allowed.
  • Simple Queue Limits: Once full, cannot insert even if front is empty (solved by Circular Queue).

Complexity Analysis

Operation Time Complexity Description
Enqueue O(1) Adding element to the rear.
Dequeue O(1) Removing element from the front.
Front/Rear O(1) Accessing the front or rear element.
Search O(n) Scanning elements.

Code Implementation

Implementation of a simple queue.

C++
#include <queue>
using namespace std;

int main() {
    queue<int> q;
    // Enqueue
    q.push(10);
    q.push(20);

    // Dequeue
    q.pop(); // Removes 10

    // Front
    int front = q.front(); // 20
    return 0;
}
Java
import java.util.LinkedList;
import java.util.Queue;

public class QueueDemo {
    public static void main(String[] args) {
        Queue<Integer> q = new LinkedList<>();
        
        q.add(10);
        q.add(20);
        
        q.remove(); // Removes 10
        
        System.out.println(q.peek()); // 20
    }
}
Python
from collections import deque

q = deque()

# Enqueue
q.append(10)
q.append(20)

# Dequeue
q.popleft() # Removes 10

# Front
front = q[0] # 20

Applications

  • CPU Scheduling (ready queue).
  • Breadth-First Search (BFS) in graphs.
  • Handling interrupts in operating systems.
  • Print spooling (tasks waiting for printer).
