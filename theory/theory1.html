Array Data Structure

Detailed explanation, analysis, and implementation.

What is an Array?

An Array is a fundamental linear data structure that collects elements of the same data type and stores them in contiguous memory locations. This contiguous storage allows for constant-time access to any element if its index (position) is known.

Arrays are the building blocks for more complex data structures like heaps, hash tables, and matrices. The memory address of the first element is often called the base address.

Types of Arrays

1. One-Dimensional Array

A linear list of elements. It needs only one subscript (index) to access an element. Example: arr[5].

2. Multi-Dimensional Array

Arrays of arrays. The most common is the 2D array (matrix), which is organized in rows and columns and requires two indices to access an element. Example: matrix[3][3].

3. Dynamic Array

Unlike standard (static) arrays which have a fixed size, dynamic arrays (like ArrayList in Java or vector in C++) can automatically resize themselves when elements are added or removed.

Advantages & Disadvantages

Advantages

  • Fast Access: Elements can be accessed in O(1) time using the index.
  • Memory Efficiency: Less overhead compared to linked lists (no pointers).
  • Cache Friendliness: Contiguous memory allocation leads to better cache locality.

Disadvantages

  • Fixed Size: Static arrays cannot change size after declaration.
  • Costly Insert/Delete: Inserting or deleting in the middle requires shifting elements (O(n)).
  • Memory Waster: If declared size is large but usage is low, memory is wasted.

Complexity Analysis

Understanding the efficiency of array operations is crucial for algorithm design.

Operation Time (Best) Time (Worst) Description
Access O(1) O(1) Direct access using index formula.
Search O(1) O(n) Linear search scans all elements. Binary search (sorted) is O(log n).
Insertion O(1) O(n) O(n) at beginning/middle due to shifting; O(1) at end (if space exists).
Deletion O(1) O(n) Similar to insertion, requires shifting to fill the gap.

Code Implementation

Below are examples of basic array operations in popular programming languages.

C++
#include <iostream>
using namespace std;

int main() {
    // 1. Declaration and Initialization
    int arr[5] = {10, 20, 30, 40, 50};

    // 2. Updating an element (O(1))
    arr[2] = 99; // Changes 30 to 99

    // 3. Traversal (O(n))
    cout << "Array Elements: ";
    for (int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}
Java
public class ArrayDemo {
    public static void main(String[] args) {
        // 1. Declaration
        int[] arr = {10, 20, 30, 40, 50};

        // 2. Accessing Length
        System.out.println("Size: " + arr.length);

        // 3. Traversal
        for (int i : arr) {
            System.out.print(i + " ");
        }
    }
}
Python
# 1. Python Lists serve as arrays
arr = [10, 20, 30, 40, 50]

# 2. Appending (Dynamic resizing)
arr.append(60)

# 3. Traversal
for item in arr:
    print(item, end=" ")

Applications

  • Used to implement other data structures like Stacks, Queues, Heaps, and Hash Tables.
  • Matrices and image processing (2D arrays of pixels).
  • CPU scheduling/Management in Operating Systems.
  • Lookup tables in dynamic programming.
