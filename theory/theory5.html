Graph Data Structure

Detailed explanation, analysis, and implementation.

What is a Graph?

A Graph is a non-linear data structure consisting of Nodes (also called vertices) and Edges. The edges connect any two nodes in the graph.

Graphs are used to represent networks. The nodes represent entities, and edges represent relationships between them.

Types of Graphs

1. Directed Graph

Edges have a direction (A -> B). You can go from A to B, but not necessarily B to A.

2. Undirected Graph

Edges have no direction (A - B). You can go both ways.

3. Weighted Graph

Edges have weights (values) associated with them, representing cost, distance, etc.

Graph Representations

Adjacency Matrix

A 2D array of size V x V where V is the number of vertices. matrix[i][j] = 1 if there is an edge from i to j.

Pros: O(1) to check if edge exists.

Cons: O(V^2) space, sparse graphs waste space.

Adjacency List

An array of lists. array[i] contains a list of all vertices adjacent to vertex i.

Pros: O(V+E) space.

Cons: O(V) to check if edge exists.

Common Algorithms

Algorithm Time Complexity Description
BFS (Breadth First Search) O(V + E) Explore neighbor nodes first, before moving to next level neighbors.
DFS (Depth First Search) O(V + E) Explore as deep as possible along each branch before backtracking.
Dijkstra's Algorithm O(E + V log V) Finds shortest path in weighted graphs.

Code Implementation

Adjacency List representation.

C++
#include <vector>
using namespace std;

int main() {
    int V = 5;
    vector<vector<int>> adj(V);

    // Add edge 0-1
    adj[0].push_back(1);
    adj[1].push_back(0);

    // Add edge 0-4
    adj[0].push_back(4);
    adj[4].push_back(0);
    
    return 0;
}
Java
import java.util.ArrayList;

public class GraphDemo {
    public static void main(String[] args) {
        int V = 5;
        ArrayList<ArrayList<Integer>> adj = new ArrayList<>(V);
        
        for (int i = 0; i < V; i++)
            adj.add(new ArrayList<Integer>());

        // Edge 0-1
        adj.get(0).add(1);
        adj.get(1).add(0);
    }
}
