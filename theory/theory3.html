Stack Data Structure

Detailed explanation, analysis, and implementation.

What is a Stack?

A Stack is a linear data structure that follows the LIFO (Last In, First Out) principle. This means that the last element added to the stack is the first one to be removed.

Imagine a stack of plates: you place (push) a new plate on top, and you can only remove (pop) the top plate.

Core Operations

1. Push

Adds an item to the top of the stack. If the stack is full, an overflow condition occurs.

2. Pop

Removes the item from the top of the stack. If the stack is empty, an underflow condition occurs.

3. Peek / Top

Returns the top element without removing it.

4. isEmpty

Checks if the stack is empty.

Advantages & Disadvantages

Advantages

  • Management: Efficient management of data in LIFO order.
  • Memory: Easy to manage memory (automatic allocation/deallocation in function calls).
  • Fast Operations: Push and Pop are O(1).

Disadvantages

  • Limited Access: Cannot access arbitrary elements; only the top is accessible.
  • Size Limit: Static stacks have fixed size (Risk of Stack Overflow).

Complexity Analysis

Operation Time Complexity Description
Push O(1) Constant time to add to top.
Pop O(1) Constant time to remove from top.
Peek O(1) Constant time to view top.
Search O(n) Linear time to find an element within the stack.

Code Implementation

Implementation using Array.

C++
#include <stack>
using namespace std;

int main() {
    stack<int> s;
    // Push
    s.push(10);
    s.push(20);

    // Pop
    s.pop(); // Removes 20

    // Top
    int top = s.top(); // 10
    return 0;
}
Java
import java.util.Stack;

public class StackDemo {
    public static void main(String[] args) {
        Stack<Integer> s = new Stack<>();
        
        s.push(10);
        s.push(20);
        
        s.pop(); // Removes 20
        
        System.out.println(s.peek()); // 10
    }
}

Applications

  • Function Call Stack (Recursion).
  • Expression Evaluation (Infix to Postfix).
  • Undo/Redo features in text editors.
  • Backtracking algorithms (Maze solving).
  • Browser History (Back button).
