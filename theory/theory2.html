Linked List Data Structure

Detailed explanation, analysis, and implementation.

What is a Linked List?

A Linked List is a linear data structure where elements are not stored at contiguous memory locations. The elements in a linked list are linked using pointers.

Each element in a linked list is called a Node. Each node generally consists of two parts:

  • Data: The value stored in the node.
  • Next: A pointer (or reference) to the next node in the sequence.

Types of Linked Lists

1. Singly Linked List

Each node contains only one pointer to the next node. Allows traversal in one direction only (forward).

2. Doubly Linked List

Each node contains two pointers: one to the next node and one to the previous node. Allows traversal in both directions.

3. Circular Linked List

The last node points back to the first node instead of NULL, forming a circle. Can be singly or doubly linked.

Advantages & Disadvantages

Advantages

  • Dynamic Size: Can grow or shrink at runtime, no need to allocate fixed size.
  • Efficient Insertion/Deletion: adding/removing elements is O(1) if the pointer is known (no shifting needed).
  • Memory Utilization: Allocates memory as needed.

Disadvantages

  • No Random Access: Cannot access elements by index (must traverse). Access is O(n).
  • Extra Memory: Requires extra memory for storing pointers/references.
  • Cache: Not cache-friendly due to non-contiguous memory.

Complexity Analysis

Operation Time (Average/Worst) Description
Access O(n) Must traverse from head to reach the k-th element.
Search O(n) Using linear search traversal.
Insertion (Head) O(1) Just update head pointer.
Insertion (Tail) O(n) O(n) if no tail pointer is maintained; O(1) if tail pointer exists.
Deletion (Head) O(1) Update head to head.next.

Code Implementation

Basic implementation of a Singly Linked List node.

C++
struct Node {
    int data;
    Node* next;

    Node(int val) {
        data = val;
        next = nullptr;
    }
};

// Connecting nodes
Node* head = new Node(10);
head->next = new Node(20);
Java
class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
        this.next = null;
    }
}
Python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

# Creating a node
head = Node(10)
head.next = Node(20)

Applications

  • Implementation of Stacks and Queues.
  • Undo functionality in software (managing states).
  • Image viewer galleries (Previous/Next images).
  • Music players (Playlists).
  • Hash tables (handling collisions via chaining).
